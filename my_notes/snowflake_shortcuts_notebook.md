| Notebook Function | Shortcut |
|---|---|

| Run cell and advance | `Shift` + `Enter` |
| Run all cells | `CTRL` + `Shift` + `Enter` |
| Stop all cells | `I` + `I` |
| Enter selected cell | `Enter` |
| Exit selected cell | `Escape` |
| Add cell above | `A` |
| Add cell below | `B` |
| Move cell up | `CTRL` + `Shift` + `↑` |
| Move cell down | `CTRL` + `Shift` + `↓` |
| Duplicate cell | `CTRL` + `D` |
| Delete selected cell | `D` + `D` |
| Delete selected cell | `Backspace` |
| Delete selected cell (skip prompt) | `Shift+D` + `Shift+D`|
| Delete selected cell (skip prompt) | `Shift` + `Backspace`|
| Convert to Markdown cell | `M` |
| Convert to Python/SQL cell | `Y` |
| Format query | `CTRL` + `Shift` + `O` |
| Comment out code | `CTRL` + `/` |
| Go to next cell | `J` |
| Go to previous cell | `K` |
| Undo | `CTRL` + `Z` |
| Redo | `CTRL` + `Shift` + `Z` |
